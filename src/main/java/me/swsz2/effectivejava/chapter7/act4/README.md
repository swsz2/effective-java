# Override 어노테이션을 일관되게 사용하라.
## 장점
    - Overload로 인한 오류를 방지할 수 있다.
    - 잘못 작성했을 경우 컴파일러가 경고를 보낸다.
### Override
    - 상속 관계에 있는 클래스간에 같은 이름의 메서드를 정의하는 것
### Overload
    - 두 메서드가 다른 이름을 갖고 있으나 인자의 수나 자료형이 다른 경우
